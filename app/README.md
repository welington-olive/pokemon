# Pokemon App

Aplicativo React Native Expo para gerenciamento de autenticação e listagem de pokemons.

## Tecnologias

- React Native Expo
- TypeScript
- React Navigation
- React Native Paper
- NativeWind (Tailwind CSS)
- Axios
- React Hook Form
- Zod
- AsyncStorage

## Instalação

```bash
yarn install
```

## Executar

```bash
yar start
```

## Estrutura

- `src/components/` - Componentes reutilizáveis
- `src/screens/` - Telas da aplicação
- `src/navigation/` - Configuração de navegação
- `src/contexts/` - Contextos React
- `src/hooks/` - Hooks customizados
- `src/services/` - Serviços de API
- `src/theme/` - Tema (cores e textos)
- `src/schemas/` - Schemas de validação Zod
- `src/types/` - Tipos TypeScript
- `src/config/` - Configurações (API, etc)

## API

O app consome a API em `https://pokemon-bu5t.onrender.com`. Certifique-se de que o backend está rodando.
